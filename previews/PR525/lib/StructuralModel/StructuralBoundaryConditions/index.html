StructuralBoundaryConditions

ONSAS.StructuralBoundaryConditions — Module

Module defining the structural boundary conditions assigned to each entitiy. This data type includes:

  • AbstractBoundaryConditions assigned to Nodes
  • AbstractBoundaryConditions assigned to Faces
  • AbstractBoundaryConditions assigned to Elements
source
ONSAS.StructuralBoundaryConditions.StructuralBoundaryCondition — Type

Structural boundary conditions. A StructuralBoundaryCondition is a collection of BoundaryConditions defining the boundary conditions of the structure.
