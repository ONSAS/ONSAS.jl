Materials

ONSAS.Materials — Module

Module defining the materials implemented. Overall, each materials consists of a data type with a label, parameters and a density into its fields.

source
ONSAS.Materials.AbstractMaterial — Type

Abstract supertype for all material models.

An AbstractMaterial object facilitates the process of defining new material models. Different material models leads to different constitutive laws, internal forces and stiffness matrices.
