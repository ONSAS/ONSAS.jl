AbstractLinearElasticMaterial Interface

This section describes the interfaces for different linear elastic material types.

ONSAS.Materials.LinearElasticMaterials.AbstractLinearElasticMaterial — Type

Abstract supertype for all elastic material models.

An AbstractLinearElasticMaterial object facilitates the process of using elastic materials.

Common methods:

  • elastic_modulus
  • poisson_ratio
  • shear_modulus
  • lamé_parameters

Common fields:

  • label
  • ρ(density)

ONSAS.Materials.LinearElasticMaterials.IsotropicLinearElastic — Type

IsotropicLinearElastic material struct.

Fields:

  • E – Elasticity modulus.
  • ν – Poisson's ratio.
  • ρ – density (nothing for static cases).
  • label – material label.
