StructuralEntities

ONSAS.StructuralEntities — Module

Module defining structural entities properties to then construct an structure. This used to create a mesh with GMSH, and the structural entities contain the ONSAS element type to the posterior conversion to the ONSAS mesh type.

ONSAS.StructuralEntities.StructuralEntity — Type

Struct used to define meshes via GMSH. Consists of a collection of element and face types, each assigned to a corresponding vector of Faces and Elements.
