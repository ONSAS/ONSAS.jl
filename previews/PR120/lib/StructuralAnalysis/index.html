Structural Analysis Interface

This section describes the interfaces to handle with different structural analysis.

ONSAS.StructuralAnalyses — Module

Module defining structural analyses that can be solved. Each structural analysis consists of a data type with an structure to be analyzed, analysis parameters and an specific state which describes the current state of structure in the analysis.

source
ONSAS.StructuralAnalyses.AbstractStructuralAnalysis — Type

Abstract supertype for all structural analysis.

An AbstractStructuralAnalysis object facilitates the process of defining an structural analysis to be solved.

Common methods:

  • structure

  • free_dofs

  • structural_state

  • initial_time

  • current_time

  • final_time

  • current_state

  • current_iteration

  • _next!

  • is_done
