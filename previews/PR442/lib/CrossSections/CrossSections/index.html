Cross sections

ONSAS.CrossSections — Module

Module defining geometric entities interface. Each cross section consists of a data type with one or more geometric parameters into its fields.

source
ONSAS.CrossSections.AbstractCrossSection — Type

Abstract supertype for all elements corss-section.

Common methods

The following functions must be implemented for a new cross-section:

    • area – Return the cross-section area.
    • Ixx – Return the moment of inertia with respect to x axis.
    • Iyy – Return the moment of inertia with respect to y axis.
    • Izz – Return the moment of inertia with respect to z axis.
    • Ixy – Return the product moment of area respect to the x-y axes.
    • Ixz – Return the product moment of area respect to the x-z axes.
    • Iyz – Return the product moment of area respect to the y-z axes.
    • I_tensor_xyz – Return the inertia tensor in the system of coordinates x-y-z.
