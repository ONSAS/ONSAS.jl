Boundary conditions

ONSAS.BoundaryConditions — Module

Module defining the boundary conditions implemented. Two types of boundary conditions are defined Load (Neumann), and Displacements (Dirichlet). Overall, each boundary condition consists of a data type with a label, dofs and values into its fields.

source
ONSAS.BoundaryConditions.AbstractBoundaryCondition — Type

Abstract supertype for all elements.

An AbstractBoundaryCondition object facilitates the process of defining:

- Dirichlet or displacements boundary conditions.
- Neumann or load boundary conditions.
