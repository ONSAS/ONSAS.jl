ONSAS.Searches.evaluate_points_in_elements — Method

Return indexes checking if a vector of points is inside a vector of elements with a serial searching algorithm. Two integer vectors are returned, the first one contains the indexes of the points inside the mesh, the second one contains the indexes of the elements containing the points.
